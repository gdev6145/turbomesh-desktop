Node Map — What to add?

The graph already exists. Select the enhancements you want. (multi-select)

A

Pan & Zoom

Scroll to zoom, drag to pan across the canvas — essential for large networks with many nodes

B

Draggable Nodes

Click and drag individual nodes to manually reposition them — pin them in place after layout settles

C

Peer-to-Peer Edges

Currently only local→peer edges are drawn. Show peer↔peer connections with hop count labels on edges

D

Packet Flow Animation

Animated dots travel along edges when messages are sent — makes routing paths visually obvious

E

Node Detail Panel

Click a node to open a sidebar with full stats: address, RSSI history chart, last-seen, hop count, send message button

F

Filter & Highlight

Search bar to highlight a specific node, dim others — useful when the network has 10+ nodes
