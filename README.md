# TurboMesh Desktop

## Overview
TurboMesh Desktop is a powerful tool designed for mesh processing and editing. This project aims to provide a user-friendly interface and integrate state-of-the-art algorithms for efficient mesh handling.

## Build and Run Instructions
To build and run the project locally, please follow these instructions:

1. Ensure you have [Java 17](https://www.oracle.com/java/technologies/javase-jdk17-downloads.html) installed on your machine.
2. Clone the repository:
   ```bash
   git clone https://github.com/gdev6145/turbomesh-desktop.git
   cd turbomesh-desktop
   ```
3. Use Gradle to build the project:
   ```bash
   ./gradlew build
   ```
4. Run the application:
   ```bash
   ./gradlew run
   ```
